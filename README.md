# React Starter Kit
> This is a boilerplate for React 16 & Webpack 4
---

## Quick Start

``` bash
# Install dependencies
npm install

# Serve on localhost:3000
npm start

# Build for production
npm run build

# Format code
npm run format
```
## App Info

### Features

 - React Router
 - Babel
 - Eslint + Prettier
 - Prettier pre-commit hook with Husky
 - SASS Loader
 - Webpack Dev Server
 - Production Build
